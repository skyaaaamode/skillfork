# Todo List Manager

A skill for creating and managing todo lists.

## Features
- Create new todo lists
- Add items to existing lists
- Mark items as completed
- Remove items from lists
- View all todo lists and their items

## Usage
To use this skill, simply ask for todo list functionality, such as:
- "Create a new todo list"
- "Add an item to my todo list"
- "Mark an item as completed"
- "Show me my todo lists"

## User Consent
Before performing any file operations or external requests, this skill will ask for your explicit consent. This includes:
- Reading or writing todo list files
- Making any network requests
- Running any scripts or commands

## Examples
**Example 1:**
Input: "Create a new todo list called 'Shopping'"
Output: "Todo list 'Shopping' created."

**Example 2:**
Input: "Add 'Milk' to my Shopping list"
Output: "Added 'Milk' to the Shopping list."

**Example 3:**
Input: "Mark 'Milk' as completed in my Shopping list"
Output: "Marked 'Milk' as completed in the Shopping list."

## Guidelines
- Be clear and specific when asking for todo list operations
- The skill will ask for confirmation before making any changes to your todo lists
- Todo lists are stored locally and will be available for future sessions
